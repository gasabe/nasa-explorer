# 🚀 NASA Explorer App

SPA desarrollada con React + Vite que consume la API pública de la NASA para explorar contenido espacial de forma interactiva.

---

## 🎯 Objetivo

Construir una aplicación que permita:

- Ver la Astronomy Picture of the Day (APOD)
- Buscar imágenes espaciales por palabra clave
- Visualizar el detalle de cada imagen
- Manejar estados: loading, error y empty
- Aplicar buenas prácticas de desarrollo

---

## 🛠️ Tecnologías utilizadas

- React
- Vite
- React Router
- Fetch API o Axios
- Custom Hooks
- MUI (opcional)

---

## ✨ Features

- Imagen astronómica del día
- Búsqueda de imágenes por keyword
- Vista de detalle
- Loading states
- Manejo de errores
- Estado sin resultados
- Código modular y reutilizable
